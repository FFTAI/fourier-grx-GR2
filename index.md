# 欢迎来到 Fourier-GRX SDK 开发文档

## 介绍

Fourier-GRX SDK 是傅利叶智能提供的一个软件开发工具包，
提供了一套API用于安装、配置、控制傅利叶智能的通用机器人产品的控制程序。

该SDK设计简洁易用，提供高级接口来控制机器人运动并读取传感器数据。
SDK 目前主要支持 Python 语言二次开发使用。
